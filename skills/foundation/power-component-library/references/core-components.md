# 核心组件清单

## 组件分类

### 基础布局组件
- **PcContainer** - 容器组件，用于页面布局和组织

### 对话框组件
- **PcDialog** - 通用对话框组件，支持自定义内容和交互
- **PcTreeDialog** - 树形选择对话框，用于选择树形结构数据

### 表单组件
- **pc-form-designer** - 可视化表单设计器，支持拖拽式表单构建
- **pc-form-render** - 表单渲染器，用于展示和处理表单数据

### 表格组件
- **pc-table** - 功能强大的表格组件，支持排序、筛选、分页等
- **pc-table-warp** - 表格包装组件，提供更多高级功能
- **PcPagination** - 分页组件，用于表格和列表的分页控制

### 数据处理组件
- **PcImport** - 数据导入组件，支持Excel等文件导入
- **PcUpload** - 文件上传组件，支持多文件上传和进度显示

### 树形组件
- **PcTree** - 树形结构组件，用于展示层级数据

## 组件详细信息

### PcDialog
**功能**：提供模态对话框，支持自定义标题、内容和操作按钮
**使用场景**：用于信息展示、确认操作、表单填写等场景
**关键特性**：
- 支持自定义尺寸和位置
- 支持遮罩层和关闭方式
- 支持键盘操作和ESC关闭
- 支持动画效果

### pc-table
**功能**：提供功能丰富的表格展示和交互
**使用场景**：用于数据列表展示、数据编辑、数据导出等场景
**关键特性**：
- 支持排序、筛选、分页
- 支持固定列和表头
- 支持树形表格和可编辑表格
- 支持虚拟滚动和性能优化
- 支持导出功能

### pc-form-designer
**功能**：可视化表单设计工具，支持拖拽式表单构建
**使用场景**：用于快速创建和配置表单，适用于需要灵活表单的业务场景
**关键特性**：
- 丰富的表单控件库
- 实时预览功能
- 支持表单验证
- 支持自定义组件和插件
- 支持表单配置导出和导入

### PcTree
**功能**：展示和操作树形结构数据
**使用场景**：用于组织架构、菜单导航、分类选择等场景
**关键特性**：
- 支持展开/折叠操作
- 支持节点选择和复选
- 支持拖拽排序
- 支持异步加载

### PcImport
**功能**：提供数据导入功能，支持Excel等文件格式
**使用场景**：用于批量数据导入、数据迁移等场景
**关键特性**：
- 支持多种文件格式
- 支持数据预览和验证
- 支持错误提示和处理
- 支持批量导入

### PcUpload
**功能**：提供文件上传功能，支持多文件和大文件上传
**使用场景**：用于头像上传、文档上传、图片上传等场景
**关键特性**：
- 支持多文件上传
- 支持上传进度显示
- 支持文件类型和大小限制
- 支持拖拽上传

## 组件选择指南

### 根据场景选择组件
- **数据展示**：使用 pc-table、PcTree
- **用户输入**：使用 pc-form-designer、pc-form-render
- **信息提示**：使用 PcDialog
- **文件处理**：使用 PcUpload、PcImport
- **页面布局**：使用 PcContainer

### 性能考虑
- 大数据量表格：使用 pc-table 的虚拟滚动功能
- 复杂表单：使用 pc-form-designer 的组件懒加载
- 频繁交互：使用防抖和节流优化

### 最佳实践
- 组件组合：根据业务需求组合使用多个组件
- 响应式设计：考虑不同屏幕尺寸的适配
- 可访问性：确保组件符合无障碍标准
- 国际化：支持多语言场景

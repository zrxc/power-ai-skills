# 上传导入导出技能
用于生成企业常见的上传、导入、导出、模板下载和文件处理流程。

## 适用场景

- Excel 导入
- 文件上传
- 模板下载
- 列表导出
- Blob 下载和文件预览前置处理

## 按需读取的参考文件

- 上传模板：`references/templates.md`
- 导入模板：`references/templates.md`
- 导出模板：`references/templates.md`

## 必须遵循的项目模式

### 上传

- 文件上传优先复用企业上传组件或已有请求封装
- 明确 `multipart/form-data`
- 上传成功后刷新列表或回填字段

### 导入

- 导入前提供模板下载
- 导入结果给出成功 / 失败条数反馈
- 导入过程需要 `loading` 和错误处理

### 导出

- 导出前校验筛选条件或选中项
- 导出接口明确 `blob` 类型
- 导出成功后提示并触发浏览器下载

## 实现清单

1. 确定是上传、导入还是导出
2. 定义接口函数和响应类型
3. 页面接入上传或导出按钮
4. 补充模板下载、成功反馈、失败反馈

## 何时切换到其他技能

- 当当前阻塞点变成接口契约、文件接口参数和返回类型时，切换到 `api-skill`
- 当页面主体是列表页，而上传或导出只是列表操作的一部分时，切换到 `basic-list-page`
- 当主要复杂度在成功失败提示、确认文案和异常反馈时，切换到 `message-skill`
- 当任务已经进入文件预览、附件展示和多格式查看，而不是上传下载链路时，切换到 `file-preview-skill`

## 注意事项

- 大文件场景关注接口超时和分片策略
- 导出接口要避免一次性无条件导出大数据
- 导入后要有清晰的异常明细反馈方案

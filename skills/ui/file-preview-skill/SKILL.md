# 文件预览技能
用于生成企业附件预览和文件查看流程。

## 适用场景

- 附件列表预览
- 图片、PDF、Office 文件预览
- 详情页附件查看
- 预览弹窗和下载联动

## 按需读取的参考文件

- 预览弹窗模板：`references/templates.md`
- 文件地址规则：`references/rules.md`

## 必须遵循的项目模式

### 数据

- 文件列表数据和业务详情数据分开维护
- 文件地址、预览地址、下载地址统一组装

### 交互

- 点击文件名进入预览或下载
- 图片和文档预览区分不同展示方式
- 大文件和不可预览文件提供回退下载方案

## 实现清单

1. 明确文件类型和预览能力
2. 实现预览地址生成
3. 接入预览弹窗或独立预览组件
4. 补充下载兜底

## 何时切换到其他技能

- 当页面主体是详情页编排，而附件预览只是其中一个区块时，切换到 `detail-page-skill`
- 当核心复杂度在上传、导入、导出、批量下载或文件处理链路时，切换到 `upload-import-export`
- 当需要弹窗交互、预览确认或关闭回刷链路时，补充 `dialog-skill`
- 当阻塞点变成文件地址拼装规则、接口返回结构或权限控制时，补充 `api-skill`

## 注意事项

- 不可预览文件必须有下载兜底
- 文件地址处理要统一封装
- 大文件预览要考虑性能和超时

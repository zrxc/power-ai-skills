# 详情页技能
用于生成符合企业页面风格的详情展示页。

## 适用场景

- 记录详情页
- 配置详情页
- 审批单据详情
- 列表页跳转详情
- 弹窗内只读详情

## 按需读取的参考文件

- 详情页模板：`references/templates.md`
- 区块划分规则：`references/rules.md`

## 必须遵循的项目模式

### 布局

- 页面使用 `PcContainer` 或企业详情容器模式
- 基础信息、扩展信息、操作记录分区展示
- 长详情页支持分块和锚点导航

### 数据

- 详情数据和列表数据分开维护
- 页面进入时按 `id` 拉取详情
- 字段展示前做好格式化和兜底

### 行为

- 详情页可包含返回、编辑、导出等操作
- 只读模式不复用编辑表单时，优先做展示组件化

## 实现清单

1. 拆分详情区块
2. 定义详情接口和详情数据结构
3. 实现详情加载与字段格式化
4. 根据需要接入编辑、导出、日志等附加区域

## 何时切换到其他技能

- 当详情页本质上是表单查看态，且字段校验、回显和禁用状态才是主要复杂度时，切换到 `form-skill`
- 当详情页里包含完整审批流、审批动作和节点状态展示时，切换到 `approval-workflow-skill`
- 当当前任务首先是列表页骨架和详情跳转链路，而不是详情展示区块本身时，切换到 `basic-list-page`
- 当阻塞点变成导出、附件预览或文件下载链路时，补充 `upload-import-export` 或 `file-preview-skill`

## 注意事项

- 长详情页避免一个模板堆满所有字段
- 字段展示优先用格式化函数，不在模板里直接写复杂表达式
- 生成详情页代码时补充中文注释，重点说明区块划分、详情加载流程、字段格式化规则和子区域联动关系
